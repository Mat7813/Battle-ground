# Battle-ground
## Portabilité
disponible sous Linux, Windows et MacOS

## prérequis
[SDL_image / SDL_ttf / SDL_Mixer](https://www.libsdl.org)

# compilation 

```bash
git clone https://github.com/Mat7813/Battle-ground
cd Battle-ground
make all
cd bin/
./jeu
```
